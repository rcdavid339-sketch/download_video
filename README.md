# Downloader Vidéo

Application simple pour télécharger des vidéos.

## Fonctionnalités
- Téléchargement vidéo/audio
- Qualité 720p/1080p/1440p 
- Queue de téléchargement

## Utilisation
1. Lancer l'application
2. Coller une URL
3. Télécharger avec la toucher " ctrl + c "
4.Downloarder des fichiers mp4 only pour le moment

Usage local uniquement
